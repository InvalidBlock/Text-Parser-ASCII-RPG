# Aposhes
A Text-Parser-ASCII-RPG

## Overview

This project is a hobby and learning-oriented ASCII RPG developed in C# using the .NET SDK. It focuses on a text-parser gameplay loop centered around grinding, character progression, and systemic design. The initial scope targets a single-player experience.

## Project Status

**Early Development**, core systems are being implemented.

## Goals

- Develop a modular RPG core;
- Implement combat and progression systems;
- Design a structured command parser;
- Maintain clean and scalable architecture;

## Tech Stack

- C#
- .NET SDK

## Build & Run

``` bash
dotnet build
dotnet run --project Aposhes.CLI
```
